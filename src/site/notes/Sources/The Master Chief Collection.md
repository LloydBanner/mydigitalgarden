This is a [[Sources/Halo\|Halo]] compilation of [[Sources/Halo Combat Evolved\|Halo Combat Evolved]], [[Sources/Halo 2\|Halo 2]], [[Sources/Halo 3\|Halo 3]], [[Sources/Halo ODST\|Halo ODST]], [[Sources/Halo Reach\|Halo Reach]] and Halo 4. I would say it is the best way to play these games other than the lack of splitscreen on PC.

My biggest issue with The Master Chief Collection is the lack of splitscreen on PC, this isn't an issue on Xbox. There is this great tool called [Nucleus Coop](https://nucleus-coop.github.io/) which I highly recommend which allows you to play splitscreen on PC and, also works with plenty of other games that are lacking splitscreen. There is also a great [guide](https://steamcommunity.com/sharedfiles/filedetails/?id=2077569545) for how to use it. Splitscreen can be a lot of fun for all the modes, if you want to play cooperatively or against each other.

Mod tools have been released for the Master Chief Collection and with them it looks like some interesting modded campaigns and maps are coming out. I haven't tried any yet but this is certainly something I may give a go in the future.
