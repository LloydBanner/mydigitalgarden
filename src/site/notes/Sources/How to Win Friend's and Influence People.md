The title of this book sounds like it is about manipulating people, I think it would be hard to use this book this way. I read this at university when I felt like I was struggling to fit in. The core principles are basically to be nice, thankful and friendly. It's ideas are simple and practical. It is definitely worth reading if you are in a similar situation.
