I set a new years resolution for 2025 to read more. I used to read a lot but in 2024 I think I read about four books. I set the goal of reading one book a month and this was the first one. This was an easy start to get myself into it and I actually read this all in one evening.

An interesting philosophy but one that I found flawed. It talks against wanting new things but I think experience is all there is in life. However I do agree that having a simpler life could be more fulfilling. It might be worth rereading.
