This is the most recent Hitman trilogy compiled into one game with a much smaller file size. I think this is the best way to play Hitman and this plays better than almost all the old games.

Each level is a semi open world sandbox where you can equip disguises, use gadgets (both goofy and serious), and infiltrate locations to hunt down your targets in any way you see fit. You can play and try to never been seen, or be a little less careful and both ways are great fun. There are also a few scripted assassination for each of your targets which are fun to preform.

With all the maps there is tonnes of equipment to unlock, and approaches to explore. Even if you just play through the main missions there is plenty to do, but there are also escalations which are progressively hard challenges, small challenges in each level for unlocks, and the freelancer mode which is a sort of roguelike mode, as well as timed elusive targets, and a few more.

I'm not the biggest fan of the escalations as they have you repeating the same challenges over and over and many of them don't change that much. Some can be fun but a lot feel a bit tedious. I found myself playing through a few to try and unlock the unlocks you get but this put me off the game for a while so I avoid this now.

The most fun way to play is to try and be stealthy and not get spotted, but not to load saves when you do. This encourages you to use the full sandbox and creates some fun tense moments, although the challenge of silent assassin (never being spotted) and suit only (never changing your disguise) can be fun.

Freelancer really brings out the best elements of the game for me. It doesn't punish you for being spotted and even sometimes encourages unconventional methods with its bonus challenges. I had the most fun I have had with this series in a long time trying to beat freelancer and it got me back into the game as a whole.

One thing that I wash there was is some form of level editor or more ways for fan created content. There is contracts mode that allows you to set targets and objectives on the maps created by IOI which allows for some user created content, but having a map editor would allow this game to live forever. Another thing I don't like is that the game is always online and requires an internet connection for a lot of features, I don't think this should be needed in a singleplayer game.

Overall this is an amazing game with a few minor flaws. I absolutely recommend it and it is one of my favourite immersive sims. There is loads of content, but do your research on which pack you are buying before you do.
