## Web Description
TrenchBroom is a free (GPLv3+), cross platform level editor for Quake-engine based games. It supports **Quake**, **Quake 2**, and **Hexen 2** and runs on **Windows** (10 and newer), **Mac OS X** (10.14 and newer) and **Linux**. TrenchBroom is easy to use and provides many simple and advanced tools to create complex and interesting levels with ease.

## My Use Case
Trenchbroom makes map making and modelling far easier than doing it in other tools such as [[Software Tools/Blender\|Blender]]. It also seems quite easy to use and integrate with [[Software Tools/Godot\|Godot]] using the plugin [[Software Tools/Func_Godot\|Func_Godot]].
