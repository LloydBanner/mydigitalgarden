Written: 11-12-2023

A lot of the performance issues seem to be fixed although in open areas there are slow downs and frame drops.  
  
I'm still encountering small bugs like the charged shot not always working and the EA launcher breaking stopping me from running the game without reinstalling it.  
  
If these issues were fixed, the game would be incredibly good. It expands on all the elements of the [[Sources/Jedi Fallen Order\|Jedi Fallen Order]] in good ways, and your power carries on from the first game. This means you feel more like a Jedi with powers like force push and pull from the start. The story is also one of the best I've seen in a [[Sources/Star Wars\|Star Wars]] game.  
  
I still have a few small gripes with how they script the climbing and restrict movement in some areas. The combat feels better but it still occasionally has weird timing on parries and attacks too.  
  
Overall a fun game that is absolutely worth playing but with the bugs and performance issues I would not buy it at the £60 it currently is a full price. It is worth it if you can get it on sale though.
