Written: 01-04-2018

A strong sequel to the first game, expanding on previous mechanics. The story continues from where the last game ended and you can transfer your save from the last game. You can choose a predefined set of characters, but I strongly recommend playing the first game first.

When a skill is maxed out you can now upgrade it to give your characters special abilities, Some cut scenes are now animated in much greater detail. However, aside from this the game is very similar to the first requiring the player to make choices which define how the story plays out.

If you didn't like the first game you won't like this, but if you enjoyed the first you will probably enjoy this more than the last game.
