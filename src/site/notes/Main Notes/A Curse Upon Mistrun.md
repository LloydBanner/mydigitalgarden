This one is medieval fantasy. The unique element is that it is set on a land which is on a giant rock that is floating through a void filled with dense mist. This land moves following a repeating cycle and occasionally meets other lands. The different houses are all on edge after a recent civil war and mythical monsters start to return causing dismay wherever they go.
