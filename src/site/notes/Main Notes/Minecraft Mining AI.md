This one I made after watching some [Codebullet](https://www.youtube.com/@CodeBullet) videos in December 2018. He did a lot of AI to automate things in games and I wanted to do something similar. The idea was to automate strip mining in [[Sources/Minecraft\|Minecraft]]. It is quite difficult to control a game without actually being able to access the game code. I suspect this would have been easier if I had done some image recognition. In the end I didn't really see the point in this because it just removed some of the fun of the game.
