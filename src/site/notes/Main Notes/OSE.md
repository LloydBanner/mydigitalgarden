[Old School Essentials](https://necroticgnome.com/pages/about-old-school-essentials) (OSE) is an old school [[Main Notes/TTRPG\|TTRPG]] based of the old Dungeons and Dragons (D&D) basic expert rules (BX).

Since running my first session of OSE on 17-05-2024 this has been my main [[Main Notes/TTRPG\|TTRPG]]. I love the simplicity of the rules, the speed of play and found it very smooth to run as a game master.
