Godot is a free open source game engine which allows 2D and 3D development. It uses a language called [[Software Tools/GDScript\|GDScript]] for scripting which is a language built for Godot to be able to make lightweight scripts. C Sharp can also be used.
## Useful Links
- [Website](https://godotengine.org/)
- [Documentation](https://docs.godotengine.org/en/stable/index.html)
- [GDscript Learning Tool](https://gdquest.github.io/learn-gdscript/?ref=godot-docs)
- [Godot University Playlist](https://www.youtube.com/playlist?list=PLrT2fbyJrAIctd7zNUsdPakIllX2lhrzo)

## Hotkeys
- F2 = rename
- ctrl + s = save
- F6 = run game
- y = snap to grid
